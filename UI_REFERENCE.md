## Color Scheme

```
Primary Blue:     #3B82F6 (Large buttons, CTAs)
Background:       #FFFFFF (Clean white)
Card Background:  #FFFFFF (White with subtle shadow)
Text:             #0F172A (Dark gray, high contrast)
Muted Text:       #64748B (Gray for help text)
Border:           #E2E8F0 (Light gray)
Success Green:    #10B981 (Checkmarks, positive)
Warning Orange:   #F59E0B (Alerts, warnings)
Error Red:        #EF4444 (Errors, destructive actions)
```

## Typography

```
Headings (H1):    48px, Bold
Headings (H2):    36px, Bold
Headings (H3):    24px, Semibold
Body:             16px, Regular
Small:            14px, Regular
Tiny:             12px, Regular

Line Heights:     1.5 (body), 1.2 (headings)
Font Family:      Inter (Google Font)
```

## Spacing & Layout

```
Container:        Max-width 1280px, centered
Section Padding:  64px vertical (desktop)
                  48px vertical (mobile)
Card Padding:     24px
Button Padding:   16px 32px
Gap Between:      24px (cards), 16px (elements)
Border Radius:    8px (cards), 6px (buttons)
```

## Animations

```
- Smooth scroll to quiz
- Progress bar transition (400ms)
- Button hover states (200ms)
- Modal fade in/out (300ms)
- Success checkmark animation
- Loading spinner rotation
```
