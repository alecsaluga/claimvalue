## Responsive Breakpoints

```
Mobile First Design:

< 640px (Mobile)
- Single column
- Full-width cards
- Stacked buttons
- Smaller typography

640px - 768px (Tablet)
- 2-column grid for features
- Larger touch targets
- Medium typography

768px - 1024px (Desktop)
- 4-column grid for features
- Max-width containers
- Optimized spacing

> 1024px (Large Desktop)
- Centered content (max-width)
- Enhanced spacing
- Larger typography
```
